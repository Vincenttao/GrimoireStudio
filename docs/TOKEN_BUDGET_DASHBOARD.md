## 3. 数据模型

### 3.1 TokenUsage 模型

```python
class TokenUsageRecord(BaseModel):
    timestamp: datetime
    spark_id: str
    agent_type: str  # "maestro", "character", "camera", "muse"
    model: str       # "gpt-4", "gpt-3.5-turbo", "deepseek"
    input_tokens: int
    output_tokens: int
    cost_usd: float
    chapter_id: Optional[str]
    success: bool
```

### 3.2 统计聚合

```python
class TokenStats(BaseModel):
    period_start: datetime
    period_end: datetime
    total_tokens: int
    total_cost_usd: float
    by_agent: Dict[str, int]
    by_model: Dict[str, int]
    total_sparks: int
    success_rate: float
```

### 5.2 成本优化建议

当检测到以下情况时，系统自动提供建议：
- Camera 使用量 > 40% → 建议切换到更便宜的模型
- 单次推演 > 100k tokens → 建议缩短场景或减少角色
- 连续失败率 > 20% → 建议检查模型配置
